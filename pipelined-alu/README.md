# 5-Stage Pipelined ALU

## Description
This project implements a 16-bit pipelined ALU with 5 stages.

## Pipeline Stages
- IF
- ID
- EX
- MEM
- WB

## Operations
- ADD
- SUB
- XOR
- MUL
- DIV

## Tools Used
- Verilog HDL
- QuestaSim
- Quartus Prime

## Results
Simulation verifies correct pipelined execution with latency.
