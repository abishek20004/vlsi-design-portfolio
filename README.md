# VLSI Design Portfolio

This repository showcases my hands-on RTL design and verification projects using Verilog HDL, including simulation, synthesis, and FPGA implementation.

---

## 📂 Projects

### 🔹 1. Traffic Light Controller (FSM)
- Designed using Verilog HDL
- Implemented as a Finite State Machine (FSM)
- Includes RTL design, testbench, and simulation waveform
- Verified using ModelSim / QuestaSim

---

### 🔹 2. 5-Stage Pipelined ALU
- 16-bit ALU with pipelined architecture
- Supports arithmetic and logical operations
- Designed using Verilog HDL
- Verified using QuestaSim
- Synthesized using Quartus Prime
- Demonstrates pipeline latency and stage-wise execution

---

### 🔹 3. Mod-5 Counter (FPGA Implementation)
- Verilog-based sequential counter design
- Implemented and tested on Artix 7 FPGA
- Includes simulation, RTL schematic, timing, and utilization analysis
- Successfully generated FPGA bitstream
