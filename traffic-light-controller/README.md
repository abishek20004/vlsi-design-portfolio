# Traffic Light Controller (FSM)

## Description
This project implements a traffic light controller using a Finite State Machine (FSM) in Verilog.

## Features
- 4-state FSM (S0–S3)
- Timer-based transitions
- Pedestrian signal support

## Tools Used
- Quartus Prime Lite
- ModelSim / QuestaSim

## Results
Simulation confirms correct state transitions.
